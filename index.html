This page used to exist to host the closed-source roguelike 'Roguemare'. At the time of the last update, the location to download the lastest version was still this. There's a dynamically as well as a statically linked version available. They should work on most any x86-64 Linux & BSD.

Please note: the static version is fully stand-alone, but has minor, unfixable issues arising from the use of netbsd-curses. The dynamic version requires recent versions of GNU libc and ncurses.

Bug reports & such: send a free-form e-mail to lotuskip at the well-known Google world-domination mail domain, with 'roguemare' in the subject line.
