This page exists to host the closed-source roguelike 'Roguemare'. Download the latest version (25 Sep, 2023), dynamic or static; more information enclosed. They should work on most any x86-64 Linux & BSD.
This is a 7-year-in-the-making release, and exceptionally includes some significant new features, mostly aimed to make the game a bit less frustrating. See the changelog for more details. (The proper anniversary date is Oct 2nd, but I shall be travelling at that time and therefore push an early release.) Also, the binaries are now hosted on a paid file server, so the links should actually keep working...

Please note: the static version is fully stand-alone, but has minor, unfixable issues arising from the use of netbsd-curses. The dynamic version requires libc.so.6 and libncursesw.so.6 present on your system.

Bug reports & such: send a free-form e-mail to lotuskip at the well-known Google world-domination mail domain, with 'roguemare' in the subject line. I'spose you can also open an issue here on Github.
